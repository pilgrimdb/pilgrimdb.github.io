An extension through Lucca province of the Via Bibulca, which provided a link from Modena for pilgrims to Lucca. The modern route links the Santuario of San Pellegrino in Alpe with the Via del Volto Santo, and then continues on via the shrine of Eremo di Calomini to the Via Francigena near Pietrasanta. Note that the Lucca site claims this links on the Emilia-Romagna side with the Via Nonantolana, but this is not correct.
